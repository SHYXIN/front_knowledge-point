## 生成计划

### 1. 项目创建顺序

按照用户要求的顺序创建五个项目：

1. Vue3 + TypeScript + ESLint + Prettier
2. Vue3 + ESLint + Prettier（不使用 TypeScript）
3. Vue2 + ESLint + Prettier（不使用 TypeScript）
4. React + TypeScript + ESLint + Prettier
5. React + ESLint + Prettier（不使用 TypeScript）

### 2. 每个项目的创建步骤

1. **初始化项目**：使用 Vite 或 Vue CLI 创建基础项目结构
2. **安装依赖**：添加 ESLint、Prettier 及相关插件
3. **配置文件**：

   * package.json（设置 scripts、依赖版本）

   * tsconfig.json（TS项目）

   * vite.config.ts/js 或 webpack 配置

   * .eslintrc.js（详细配置及解释）

   * .prettierrc（格式化规则）

   * .gitignore
4. **项目结构**：

   * src 目录（components、views、router、api 等）

   * 至少 2 个页面（Home 和 About）

   * 1 个组件示例

   * API 请求示例

   * 路由配置
5. **README.md**：

   * 项目介绍

   * 运行步骤

   * Windows + NVM 注意事项

   * 关键学习点

   * ESLint + Prettier 协作机制

### 3. 对比表格和关键点总结

* 生成对比表格，包含框架/版本、TS使用情况、目录结构差异、ESLint配置差异、组件写法差异、生命周期/Hook差异、Router使用差异、代码风格/格式化差异

* 关键点总结，包含TS vs JS差异、Vue2 vs Vue3差异、React + TS vs React JS差异、ESLint + Prettier配置差异和协作机制

### 4. Windows + NVM 注意事项

* 在每个项目的README中明确写明Node版本切换方法

### 5. 技术栈选择

* Vue3项目：Vite + Vue Router 4

* Vue2项目：Vue CLI + Vue Router 3

* React项目：Vite + React Router 6
